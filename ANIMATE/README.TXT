This version of A8Viewer is for Anim8or v0.8 export files.

To compile A8Viewer you need the Glut32.h header.  To link
you need to specify the OpenGL32.lib and GLut32.lib libraries.
You also need the Glut32.dll dynamic link library to run.

The OpenGL32 library comes with Microsoft C compiler.
The Glut32 files can be found through: www.opengl.org.
